# deno-routing

## prerequisite

- [deno](https://deno.land/)
- [node :p](https://nodejs.org/en/)

## setup && run

from root:

```bash
npm run setup && npm run dev
```

## Note

`client/` requires a `.env` file with at least these variables:

```
REACT_APP_NODE_ENV=dev
REACT_APP_SERVER=http://localhost:8000
```

Happy Hacking!
